# Methods text

Sequencing reads were processed using a refined Bash implementation of a WGS preprocessing and germline short-variant discovery workflow based on the GATK Best Practices framework. Raw reads were retrieved from the Sequence Read Archive or provided as local paired-end FASTQ files. Read quality was assessed with FastQC and summarized with MultiQC before and after trimming. Adapter and low-quality bases were removed with Trim Galore using a Phred quality threshold of 10 and a minimum read length of 5 bp.

Reads were aligned to the selected reference genome using BWA-MEM with read-group information specified at the alignment step. Alignments were streamed directly to SAMtools to generate coordinate-sorted BAM files, which were subsequently indexed. Alignment summaries were generated using SAMtools flagstat. The reference genome was indexed using BWA and SAMtools, and a GATK sequence dictionary was generated before downstream analyses.

Duplicate reads were marked using GATK MarkDuplicates, and duplicate metrics were retained for quality control. When an appropriate known-sites VCF was available, base quality score recalibration was performed using GATK BaseRecalibrator and ApplyBQSR, with recalibration diagnostics generated using AnalyzeCovariates. For datasets lacking reliable known polymorphic sites, BQSR was disabled. Variants were called using GATK HaplotypeCaller, either in GVCF mode for downstream joint genotyping or in standard VCF mode for single-sample analyses. When multiple GVCFs were generated, joint genotyping was performed using GenomicsDBImport and GenotypeGVCFs. Simple hard filtering was optionally applied separately to SNPs and indels.
