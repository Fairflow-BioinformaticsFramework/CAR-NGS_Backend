## Methods-style description

Raw paired-end sequencing reads were processed using a reproducible short-read variant discovery workflow based on BWA-MEM, SAMtools and GATK. Read quality was assessed before and after trimming using FastQC and summarized with MultiQC. Adapter and low-quality sequence trimming was performed with Trim Galore using a minimum Phred quality threshold of 10 and a minimum retained read length of 5 bp. Reads were aligned to the selected reference genome using BWA-MEM, with read-group information added at the alignment step. Alignments were converted directly to coordinate-sorted BAM files using SAMtools and indexed for downstream analyses.

The reference genome was indexed with BWA and SAMtools, and a GATK sequence dictionary was generated before variant discovery. Duplicate reads were marked using GATK MarkDuplicates, and duplicate metrics were retained for quality control. When a suitable set of known variants was available, base quality score recalibration was performed with GATK BaseRecalibrator and ApplyBQSR, and recalibration quality was evaluated with AnalyzeCovariates. For organisms lacking reliable known polymorphic sites, base quality score recalibration was disabled to avoid introducing unsupported corrections. Germline short variants were called using GATK HaplotypeCaller, either in GVCF mode for subsequent joint genotyping or in standard VCF mode for single-sample analyses. Pipeline execution parameters, software versions and file paths were recorded in a run manifest to improve reproducibility.
